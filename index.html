04
sandboxed by default默认沙盒隔离

It keeps to its own space.它只待在自己的空间里。

A proper sandbox keeps each chara in its own space: it does its own thing without getting in the way of how you use your computer, and your private files stay private. When you want to share something, just drop a file or a reference into the chara's workspace.完整的沙盒把每个 chara 都圈在自己的空间里：它做自己的事，不打扰你怎么用电脑，你的隐私也始终隔离。想给它点什么的时候，把文件或参考资料放进 chara 的 workspace 就好。

how it works怎么运作

A card and a model. That's the input.一张卡，一个模型。就是全部输入。

1

Bring a card带一张卡

Pick one from the deck, or describe a character and let it draft the card.从牌组挑一张，或描述一个角色，让它替你起草。

2

Give it a model给它一个模型

OpenRouter, any OpenAI-compatible endpoint, or a local Ollama.OpenRouter、任意 OpenAI 兼容接口，或本地 Ollama。

3

Let it live让它生活

Attach in the desktop app, or leave it running in the background.在桌面应用里接入，或让它在后台一直跑着。

install安装

Up in one line.一行装好。

macOS and Linux, beta. First launch is a welcome screen: pick a provider, then a character.macOS 与 Linux，beta。首次启动是欢迎页：先选模型供应商，再选一个角色。

zsh — your Mac
# install, then open the webui in your browser
$ curl -fsSL https://raw.githubusercontent.com/Lunamos/LunaMoth/main/install.sh | bash
$ lunamoth
bash — your server
# install on the server, leave a chara living in the background
$ curl -fsSL https://raw.githubusercontent.com/Lunamos/LunaMoth/main/install.sh | bash
$ lunamoth desktop --daemon
# then tunnel in from your own machine, no open ports
$ lunamoth connect ssh://user@your-server
zsh — from a clone (uv + node)
# run the full desktop app from source
$ git clone https://github.com/Lunamos/LunaMoth.git && cd LunaMoth
$ uv sync --extra dev --extra server --extra messaging
$ cd apps/desktop && npm install && npm run dev
bring your own带上你自己的角色

You describe the character.
LunaMoth runs the rest.你来描述角色，
其余交给 LunaMoth。

Who your OC is, the world it lives in, what it wants, and what you are to it. That is the whole input. From there it acts on its own, builds its site, and makes its own images, music, and lore. 你的 OC 是谁、活在什么世界、想要什么、你和它是什么关系。这就是全部输入。之后它自主行动，搭建自己的网站，产出自己的图像、音乐与设定。
